# Vision Girls Hostel

A premium MERN stack website for Vision Girls Hostel with a polished React frontend, secure Express backend, JWT authentication, and admin/resident dashboards.

## Prerequisites

- Node.js 18+ / npm
- MongoDB running locally or accessible via URI

## Setup

1. Install dependencies
   - `cd server && npm install`
   - `cd ../client && npm install`

2. Configure environment variables
   - Copy `server/.env.example` to `server/.env`
   - Copy `client/.env.example` to `client/.env`

3. Seed sample data
   - `cd server && node seed.js`

4. Start the backend
   - `cd server && npm run dev`

5. Start the frontend
   - `cd client && npm run dev`

6. Open the app
   - `http://localhost:5173`

## Project Structure

- `server/` - Express API, MongoDB models, authentication, uploads
- `client/` - Vite React app with Tailwind, pages, components, auth

## Notes

- JWT is stored in an httpOnly cookie for security.
- Multer saves uploads to `server/uploads`.
- The admin user and sample data are generated by `server/seed.js`.
